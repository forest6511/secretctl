# Security

secretctl is designed with security as the top priority.

## Key Security Features

- **AES-256-GCM** encryption for secrets at rest
- **Argon2id** key derivation with OWASP-recommended parameters
- **Option D+** security model for AI integrations
- **Local-first** architecture - your secrets never leave your machine
